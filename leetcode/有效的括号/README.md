## 题目: 有效的括号

> 栈

### 题目说明
````
给定一个只包括 '('，')'，'{'，'}'，'['，']' 的字符串，判断字符串是否有效。

有效字符串需满足：
    1. 左括号必须用相同类型的右括号闭合。
    2. 左括号必须以正确的顺序闭合。

左括号必须用相同类型的右括号闭合。
左括号必须以正确的顺序闭合。
注意空字符串可被认为是有效字符串。

来源：力扣（LeetCode）
链接：https://leetcode-cn.com/problems/valid-parentheses
著作权归领扣网络所有。商业转载请联系官方授权，非商业转载请注明出处。
````

### Example:
```
输入: "()"
输出: true

示例 2：

输入: "()[]{}"
输出: true

示例 3：

输入: "(]"
输出: false

示例 4：

输入: "([)]"
输出: false

示例 5：

输入: "{[]}"
输出: true

提示：

```

### 题解:
```
通过阅读题目后，我们可以通过借助栈来实现该题目：
# Step 1：我们遍历元素，当遇到所有左括号时候，我们将元素放入栈中，
# Step 2: 当遇到右括号的时候，我们取出栈中的元素与其匹配，如果对不上，
返回false,能对上继续往后走
# Finally: 最后通过判断栈是否为空,如果为空返回true,不为空返回false

时间复杂度O(N)
空间复杂度o(N)
```
